# AI Dev Tools Update Tracker

A modern web application that automatically tracks and displays updates from leading AI development tools:
- **Google Antigravity** - AI-powered IDE
- **Cursor** - AI coding assistant IDE  
- **Warp** - Agentic development environment/terminal
- **Claude Code** - Terminal-based agentic coding tool

## Project Structure

This is a monorepo containing:
- `apps/web` - Next.js frontend and API routes
- `apps/cloud-agent` - Oz cloud agent for automated scraping
- `packages/database` - Prisma database schema and migrations
- `packages/shared-types` - Shared TypeScript types across apps
- `packages/ui` - Shared UI components

## Tech Stack

- **Frontend**: Next.js 14+, TypeScript, Tailwind CSS (glassmorphism design)
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL (Supabase)
- **Agent**: Oz Cloud Agent running on Warp infrastructure
- **Hosting**: Vercel (frontend/backend), Warp (agent)
